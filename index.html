ゲーム概要

『神環記 : Record of the Gods』は、
森・文化・コミュニティ・空環マネー をつかって、 20年間で世界を立て直す協力型ボードゲームです。

プレイヤーたちはそれぞれ 「森の守り手」「文化の継承者」「コミュニティの灯火」「空環エンジニア」 といった神格を持つ存在として行動します。

3つの指標：

  • 🌳 森（F） Forest
  • 🎎 文化（K） Culture
  • 🤝 コミュニティ（S） Social

を育てながら、 空環ポイント（P） を使って「循環する経済」を作っていきます。

生命指数（L） は、

L = F + K + S

で計算されます。20年の終わりに、

  • L ≥ 40 … 🌈 神々の祝福エンディング
  • 30〜39 … 🌿 浄化の兆しエンディング
  • 20〜29 … 🌙 揺らぎの未来エンディング
  • 19以下 … 🔥 神々の嘆き（文明崩壊）

そしてこのゲームの裏テーマは、
「人間もまた八百万の神の一柱であることを思い出す」こと。

世界観（ストーリー）

経済と効率と利子に支配された文明は、森を削り、文化を忘れ、人のつながりを弱らせてしまいました。
神々は嘆きつつも、まだ希望を捨ててはいません。

そこで呼ばれたのが、あなたたちです。

人もまた、八百万の神々の一柱である。
ただ、その「神性」を忘れているだけなのだ。

空環（くうかん）は、使うほど循環し、貯めこむと減衰する新しい通貨。（単位は、１Akasha）
旧来の「利子のための貨幣」ではなく、 生命と調和を育てるためのマネーとして設計されています。

プレイヤーはそれぞれの神格として行動しながら、 森・文化・人のつながりを取り戻し、
自らの中に眠る「神の働き」を思い出していきます。

遊び方（ざっくり）

プレイ人数

1〜4人（4人推奨）。4人の場合、4つの神格を1人1つずつ担当します。

1年（1ターン）の流れ

  1. イベントカードを1枚めくる（災害・祭り・神々の加護・旧経済の誘惑など）
  2. プレイヤー全員で「今年どう動くか」を相談する
  3. アクションカードを最大3枚選び、それぞれの効果をボードに反映する
  4. 空環ポイント（P）の減衰（デミュレージ）を処理する
  5. F, K, S を合計して生命指数 L を更新する
  6. F=0 / K=0 / S=0 があれば即ゲームオーバー

詳しい数値やカード一覧は 📜 ルールページ にまとまっています。

コンポーネント（必要なもの）

  • A4ボード（F / K / S / P / L のトラックを描いたもの）
  • プレイヤーごとの神格カード（森・文化・コミュ・空環）
  • イベントカード（25枚）
  • アクションカード（20枚）
  • 🧩 才能カード（20枚） - プレイヤーの得意なことを表すカード。アクションに重ねてボーナス効果を発揮
  • 🏗 共創プロジェクトカード - プレイヤー全員で少しずつ進める大きな共同プロジェクト
  • コマやビーズ（トラック用）
  • そして、iPhoneでも使える簡易アプリ盤面

物理ボードとカードを手作りしつつ、点数管理だけ 📱 簡易アプリ を使うのが遊びやすい想定です。

才能カードについて

各プレイヤーはゲーム開始時に2〜3枚の才能カードを持ちます。アクションカードに相性の良い才能を重ねると、効果に+1ボーナスが乗ります。異なる才能を2枚まで重ねて、合計+2までボーナス可能です。

共創プロジェクトについて

ゲーム開始時に2〜3枚の共創プロジェクトカードを場に置きます。各ターン、プレイヤーは自分の才能カードを1枚までプロジェクトに捧げることができ、完成条件を満たすと大きな効果が発動します。

詳しいルールは📜 ルールブックを参照してください。
