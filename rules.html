1. 世界観と裏テーマ

この世界には、森・水・火・風・道・技・縁……あらゆるものに 八百万の神（やおよろずのかみ） が宿っています。

そして、人もまた八百万の神々の一柱（ひとはしら）である。
ただ、その「神性」を忘れているだけなのだ。

経済成長と利子と効率のために自然を削り、文化を捨て、人のつながりを弱らせてしまった文明は、 今や限界に近づいています。

空環（あかし）は、そんな世界で生まれた 「循環する生命のマネー」です。

  • 貯めこむと価値が減っていく（デミュレージ）
  • 森や文化やコミュニティを育てると、空環が巡ってくる

このゲームの表の目的は、 20年間で森・文化・コミュニティを立て直すこと。
裏の目的は、 「自分も神の働きを持つ存在だった」ことを思い出すことです。

各プレイヤーはそれぞれの才能を持ち寄って、森・文化・コミュニティを支え合います。
いろいろな才能が重なり合うほど、村全体の「にぎわい」と「響き」が増していきます。

2. パラメータと勝利条件

  • 🌳 森 F（Forest） 0〜20
  • 🎎 文化 K（Culture） 0〜20
  • 🤝 コミュニティ S（Social） 0〜20
  • 💫 空環 P（Akasha） 0〜20
  • 🌏 生命指数 L（Life） 0〜60

生命指数は次の式で計算します：

L = F + K + S

ゲームは 20年間（20ターン）続き、終了時の L によってエンディングが変化します。

  • L ≥ 40 … 🌈 神々の祝福エンディング
  • 30〜39 … 🌿 浄化の兆しエンディング
  • 20〜29 … 🌙 揺らぎの未来エンディング
  • 19以下 … 🔥 神々の嘆き（文明崩壊）

即時敗北条件：

  • F = 0（森が完全に失われる）
  • K = 0（文化が途絶える）
  • S = 0（コミュニティが崩壊する）

オプション：にぎわい N トラック

才能を活かした共創の度合いを表す指標として、 にぎわい N（0〜10程度）を追加してもよい。

  • 1ターンの間に、3種類以上の才能カードが使われたら N +1。
  • N が 5 に達したら：それ以降、毎ターン終了時に L +1。
  • N が 8 に達したら：悪いイベントの効果を常に −1 軽減。

※ にぎわいトラックは「才能システムで遊び慣れてきたら」追加する発展ルールとして扱う。

3. プレイヤーの神格（ロール）

4人プレイ時には、それぞれ次の役割を1つ担当します。

🌳 森の守り手（Forest Guardian）

  • 植林や里山整備など「森系アクション」でボーナス。
  • 自然災害イベントの被害を軽減できる。
  • 山の神・木の精霊に近い性質。

🎎 文化の継承者（Heritage Weaver）

  • 祭り・芸能・職人技継承など「文化系アクション」でボーナス。
  • K（文化）を伸ばし、精神的な豊かさを生む。
  • 大国主命・天宇受売命のような性質。

🤝 コミュニティの灯火（Community Keeper）

  • 助け合い・ケア・対話など「コミュニティ系アクション」でボーナス。
  • 不和イベントを和らげる。
  • 稲荷・和合の神々のような「縁」の性質。

💫 空環エンジニア（Akasha Architect）

  • 空環（P）の増減やデミュレージ無効化でボーナス。
  • AI（神託アルゴリズム）関連イベントの影響を軽減できる。
  • 少彦名命のような「智」の性質。

4. 1ターンの流れ（1年）

  1. イベントフェーズ：イベントカードを1枚めくる。
  2. 相談フェーズ：プレイヤー全員で今年の方針を話し合う。
  3. アクション選択フェーズ：
    アクションカードを最大3枚選び、「誰がどの才能を重ねるか」を相談する。
  4. アクション実行フェーズ：
    F / K / S / P を変更し、そのアクションに重ねられた才能カードを反映する。
  5. 空環デミュレージ：P が一定以上なら減衰させる（例：Pが5以上なら −1）。
  6. 生命指数更新：L = F + K + S を計算する。
  7. 敗北チェック：F, K, S のいずれかが 0 でないか確認。
  8. （オプション）にぎわい判定：1年のあいだに使われた才能の種類を数え、N を更新する。

数値の管理には 📱 簡易アプリ を使うと便利です。

5. 才能カードの使い方

プレイヤーが本来持っている「得意なこと」や「好きなこと」を表すのが、 才能カードです（🧩 talents.html を印刷）。

5-1. ゲーム開始時の準備

  • 才能カードをシャッフルし、各プレイヤーに2〜3枚ずつ配る。
  • もしくは、配られた中から 「これは自分っぽい」才能を2枚選ぶ方式にしてもよい。
  • 余った才能カードは山札として横に置き、 イベントなどで追加の才能を得られるルールを作ってもよい。

5-2. アクションとの相性ボーナス

各アクションカードには、「相性の良い才能アイコン」を 1〜2 個描いておきます。 例：

「鎮守の森 植樹祭」
相性：🌱 育てる才能 / 👐 手しごとの才能

  • そのアクションを実行するとき、
    相性に合う才能カードを持つプレイヤーが、 そのカードをテーブルに出す（このターンその才能を捧げる、というイメージ）。
  • 相性が1つ合うごとに、アクションの効果に +1 ボーナス。
    例：本来 F +2 のところを、才能1枚で F +3 にするなど。
  • 同じアクションに、異なる才能を2枚まで重ねてよい。
    2種類の才能が重なった場合、合計 +2 までボーナスをつけてもよい。
  • 才能カードは、1ターンに1回まで使える（次のターンにはまた使える）。

5-3. 実際のプレイヤーの才能を書き込むオプション

ゲーム開始前に、各プレイヤーが紙に自分の得意・好きなことを3つ書き、 それをもとにオリジナルの才能カードを手書きで作ってもよい。

例：「革細工の才能」「ドラムの才能」「Clojureで組む才能」など、 実際の自分に寄せたカードを混ぜると、かなり楽しい。

6. 共創プロジェクトカード

共創プロジェクトカード（🏗 projects.html）は、 プレイヤー全員で少しずつ進めていく 大きな共同プロジェクトを表します。

6-1. 配置と基本ルール

  • ゲーム開始時に、共創プロジェクトカードを数枚ランダムに引き、2〜3枚を表向きで場に出す。
  • 残りは裏向きの山にしておき、ゲーム中のイベントなどで追加してもよい。
  • 各ターン、プレイヤーは 自分の才能カードを1枚まで、どれか1つのプロジェクトに捧げることができる。

6-2. 完成条件と効果

プロジェクトカードには、必要な才能や条件、完成したときの効果が書かれています。

例：地域まるごと文化祭（抜粋）

  • 必要な才能例：手しごと／表現／企画／つなぐ／もてなし
  • 完成条件：違う才能カードが 4種類以上 置かれている。
  • 完成効果：K +3 / S +3 / P +2

例：多世代シェア工房（抜粋）

  • 完成条件：3ターン以上にわたり、合計5枚の才能カードが置かれる。
  • 完成効果：F +1 / K +2 / S +2 ＋「修理・再生」系アクション恒久 +1

共創プロジェクトは、短期的にはあまり数字が増えないが、完成すると大きな変化をもたらす 「長期投資」のような存在です。

7. アクションカード（例）

アクションカードは、1年のあいだにできる具体的な取り組みを表します。 才能カードとの相性アイコンを付けておくと、使うたびに物語が立ち上がります。

7-1. 森系

「鎮守の森 植樹祭」

  • 効果：F +2 / S +1 / P +1
  • 代償：K −1
  • 相性の良い才能：🌱 育てる / 👐 手しごと / 🧑‍🍳 もてなし
  • 森の守り手が関わると、さらに F +1。

「山の神への奉仕」

  • 効果：F +1（次の自然災害のF被害 −1）
  • 代償：P −1
  • 相性の良い才能：🌱 育てる / 🧵 細やかさ

7-2. 文化系

「鎮守の祭り 準備」

  • 効果：K +2 / S +1 / P +1
  • 代償：F −1
  • 相性の良い才能：💡 企画 / 🎨 表現 / 🧑‍🍳 もてなし

「職人技 継承の稽古日」

  • 効果：K +2 / S +1
  • 代償：P −1
  • 相性の良い才能：👐 手しごと / 📚 教える

7-3. コミュニティ系

「子ども食堂と見守り」

  • 効果：S +2 / P +1
  • 代償：K −1
  • 相性の良い才能：👂 聴く / 🏡 場づくり / 🧑‍🍳 もてなし

「村じゅうお掃除の日」

  • 効果：S +1 / F +1
  • 代償：なし
  • 相性の良い才能：🔥 推進 / 🧵 細やかさ

7-4. 空環・AI系

「空環アップグレードの儀」

  • 効果：P +2（次ターンのデミュレージ無効）
  • 相性の良い才能：🧮 仕組み化 / 📊 見える化 / 🧠 学び

「AI・神託アルゴリズム更新」

  • 効果：悪いイベントの被害 −1（このターン） / P −1
  • 相性の良い才能：🧠 学び / 📊 見える化 / 🛡 守り

※ 実際のアクションカードは、これらの例をベースに自由に追加・調整してよい。

8. イベントカード（方向性）

25枚ほどを想定。主なカテゴリ：

  • 自然イベント（豪雨、山火事、外来種、豊作 など）
  • 文化イベント（祭りの復活、職人の引退、伝統授業 など）
  • コミュニティイベント（奉仕、トラブル、祝い事、空き家問題 など）
  • 旧経済イベント（投機の誘惑、大規模開発、価格競争 など）
  • 神々の祝福イベント（天照、大国主、木花咲耶姫、猿田彦、少彦名 など）

「旧経済の誘惑」カードでは、P は増えるが K が大きく減るなど、 魂の試練が起きます。

9. 神性を思い出すエンディング演出（案）

ゲームが成功したとき、読み上げ用の言葉の例：

あなたたちは、森を守り、文化をつなぎ、人を結び、
忘れられていた道をもう一度歩み出した。

その姿はもはや、ただの「人」ではない。
八百万の神々の一柱としての働きを、再びこの世界に顕した者たちである。

才能カードや共創プロジェクトカードを振り返りながら、
「今回はどんな才能がどんな場面で光ったか」を語り合うと、物語がさらに深まります。
